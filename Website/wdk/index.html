Strategies Web Development Kit
Introduction

Use the Strategies WDK to create powerful "data mining" websites. It was developed for "omics" databases (genomics, proteomics, etc), but will work with any relational database system and on any schema.

Use the WDK to design:

  • a coarse grained Data Model (using your existing schema) that specifies record types and searches against them.
  • a View of that data model using JavaServer Pages

Please refer to the WDK documentation to learn more.
Sites that use the WDK

Here are some production sites powered by the WDK:

  • PlasmoDB - A genomic database for the malaria-causing parasite Plasmodium falciparum.
  • BCBC Genomics - The Beta Cell Biology Consortium Genomics database.


The WDK Template Site
The WDK Template Site is a simple but fully featured site that demonstrates the power of the WDK. It is also a template for you to create your own site from. You can download and install it (see the Access section below) and modify to your own needs. The site is completely specified by:
  • Its Model (XML)
  • Its View (JSP and CSS)
